## How to run the project?
## Pre-requisites ( must be done each time frontend or backend packages are updated )
1. ```bash
    cd src/frontend
    npm install
    ```
2. ```bash
    ... # entrer venv setup
    cd src/Backend
    pip install -r requirements.txt
    ```
## Running the project
1. ```bash
    cd src/frontend
    npm start
    ```
2. ```bash 
    ... # entrer venv setup
    cd src/Backend
    python manage.py runserver
    ```
